# YumMonsters - 이미지 폴더

이 폴더에 몬스터 이미지를 넣으세요!

## 파일 이름 규칙
파일 이름 = 도감 번호 (숫자.확장자)

예시:
- 1.png     → 도감 #000001 (불꽃 타입 1번 카드)
- 500.jpg   → 도감 #000500
- 10001.png → 도감 #010001 (물 타입 1번 카드)

## 도감 번호 계산법
도감번호 = (카테고리 번호 × 10,000) + 카드 번호 + 1

카테고리 번호:
 0=불꽃(1~10000)   1=물(10001~20000)  2=풀(20001~30000)
 3=전기            4=얼음             5=격투
 6=비행            7=어둠             8=강철
 9=드래곤         10=달콤            11=매운
12=빵             13=면              14=고기
15=채소           16=바다            17=하늘
18=숲             19=우주(190001~200000)

지원 형식: PNG, JPG, JPEG, WEBP, GIF
이미지 없으면 이모지로 자동 대체됩니다.
